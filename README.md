# Wuji Studio

[![Release](https://img.shields.io/github/v/release/wuji-technology/wuji-studio)](https://github.com/wuji-technology/wuji-studio/releases)

Wuji Studio is a visualization and device management platform for [Wuji](https://wuji.tech) robotics devices.

## Download

[Download Wuji Studio](https://github.com/wuji-technology/wuji-studio/releases) for Linux (Ubuntu 22.04+).

## Documentation

Visit the [Wuji documentation](https://docs.wuji.tech) to get started.
